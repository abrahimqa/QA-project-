2.3) Testing Types

Written below are all the testing types that we will use and plan during the STP Project about the website Aroma:

Smoke Testing - We will use this testing on the website itself.

With this testing, we will check every important and critical step in the website about enter the menu , check the aroma card and customer service……

Functional Testing - We will use this testing to see if every feature in the website works as it should be.We will test if every level of the feature works without any errors/mistakes.

Negative Testing - We will do functions unknown for the website or not correct functions  like: try to send a letter to Aroma without fill the name , last name …….writing unknown name, write in other languages, write down words that make no sense in search fields and don’t fill a search field.

Localization Testing - We will use this testing to check the website language can be change  by choosing a different language in the sittings and change the home land settings, and to check if the user can see aroma branches that located in Israel / aroma branches in Canada and Ukraine.

Usability Testing - We will use this testing to see if the UI of the website is comfortable for every person.

We will check if the website Aroma is going by order at every feature, checking if there are errors in each “Click”, if the website loads fast each action and more.
